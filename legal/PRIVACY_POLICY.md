## 7. Data Security

We implement industry-standard security measures to protect your information:

### 7.1 Technical Safeguards
- **Encryption**: All data is encrypted in transit (TLS 1.3) and at rest (AES-256)
- **Access Controls**: Row-level security (RLS) ensures users can only access their own data
- **Authentication**: Secure OAuth 2.0 via Supabase
- **Security Headers**: CSP, HSTS, X-Frame-Options protect against web attacks

### 7.2 Administrative Safeguards
- Employee access to user data is limited to essential personnel only
- All employees sign confidentiality agreements
- Regular security audits and penetration testing

### 7.3 Limitations
No security system is perfect. While we strive to protect your information, we cannot guarantee absolute security. You are responsible for maintaining the security of your account credentials.
